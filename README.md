# 🐍 Snake Water Gun Game

A console-based **Snake Water Gun** game built with **Python**. The game includes voice feedback using `pyttsx3`, score tracking, multiple rounds, and input validation.

## 📌 Features

* 🎮 Play the classic Snake Water Gun game.
* 🔊 Voice feedback using `pyttsx3`.
* 📊 Tracks wins, losses, and draws.
* 🔁 Play multiple rounds.
* ✅ Handles invalid user input gracefully.

## 🛠️ Technologies Used

* Python 3
* `random`
* `pyttsx3`

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/himangparitoshbaberwal/python-snake-water-gun.git
```

2. Open the project folder.

3. Install the required library:

```bash
pip install pyttsx3
```

4. Run the program:

```bash
python snake_water_gun.py
```

## 🎯 Game Rules

* 🐍 Snake drinks Water.
* 💧 Water extinguishes Gun.
* 🔫 Gun kills Snake.
